LDAP Group Sync

LDAP Group → Team Маппинг

Автоматически добавляет пользователей в проекты при входе через LDAP

🔗
Как работает LDAP Group Sync:
1. Пользователь входит через LDAP
2. Velum получает список его групп (атрибут memberOf)
3. Для каждой группы из маппинга — пользователь добавляется в проект с указанной ролью
4. Если пользователь уже в проекте — ничего не происходит (идемпотентно)
Укажите полный DN группы, например: CN=DevOps,OU=Groups,DC=company,DC=com
