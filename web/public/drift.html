Drift Detection

GitOps Drift Detection

Мониторинг расхождений между желаемым состоянием (IaC) и реальным состоянием инфраструктуры

📡
Как работает Drift Detection:
1. Выберите шаблон для мониторинга (Ansible/Terraform)
2. Нажмите «Проверить» — запустится задача с --check --diff
3. Результат сохраняется: clean / drifted / error
4. Настройте расписание для автоматических проверок
